# 1. Introduction

## 1.1 Overview
This project, titled "Material Processing and Inventory Management System," is designed to automate the lifecycle of raw materials as they move through various processing stages.
Database Management Systems (DBMS) are essential for maintaining data integrity and providing structured storage for large datasets like the one sourced from Kaggle.

## 1.2 Scope of the Project
The scope includes tracking material intake, monitoring processing status (Pending/In-Progress/Completed), and managing supplier information.

## 1.3 Objectives
* To create a robust database using MySQL.
* To implement CRUD (Create, Read, Update, Delete) operations for material tracking.
* To apply relational constraints and normalization to ensure data accuracy.
* To generate views and indexes for optimized performance..
