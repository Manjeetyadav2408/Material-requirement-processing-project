# 2. System Analysis

## 2.1 Existing System
The current manual or spreadsheet-based system for material processing often suffers from data redundancy, difficulty in tracking batch history, and lack of real-time status updates.

## 2.2 Proposed System
The proposed MySQL-based system provides a centralized platform to store all material data. It ensures referential integrity between materials and their suppliers.

## 2.3 Hardware & Software Requirements
### Software Requirements
* **Operating System:** Windows 10/11
* **Database:** MySQL Server
* **Tool:** XAMPP or MySQL Workbench

### Hardware Requirements
* **RAM:** Minimum 4GB
* **Storage:** 20GB HDD/SSD
